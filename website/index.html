TempoMeter

Information

Following test investigates the human ability to keep up with isochronous and gradually changing tempo.

It takes up to four minutes to complete this test.

You will see the graph of your performance in the end.

This test does not work on mobile, or tablet browsers!

This test is anonymous, however, it uses cookies to store data of your performance. We will assume you are OK with it, if you start the test.

Instructions

This test consists of three parts. Each part starts with the metronome sound. After the text on the website changes from "Focus!" to "Tap Now!", you should start tapping the button with your mouse or touchpad. The metronome sound will go away after some time, but you should keep tapping to the tempo set by the metronome to the best of your abilities until the text changes to "Stop tapping". You will be automatically redirected to a short survey after each part. The surveys have to be submitted by clicking on the "Complete" button.

Use headphones during the test! The delay created by the speed of sound traveling in the air will be reduced to minimum by doing so. Set the volume using the following test sound to a comfortable level.

While performing this test, you should be in a quiet environment!
